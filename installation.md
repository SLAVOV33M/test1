# Requirements:
1. [ ] PHP >=8.1
2. [ ] NodeJs >=18
3. [ ] npm >=9.2
4. [ ] Composer ~2


# Install
<pre>
git clone project-url
cd project-url

composer install


npm install

php artisan key:generate

php artisan serve
php artisan migrate
php artisan db:seed

</pre>

You can import existing demo db from `project-url/db/xbot-db.sql`
